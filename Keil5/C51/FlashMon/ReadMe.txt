The sub-folders contain pre-configured Monitor versions for various
Atmel 8051 device variants.

Currently the following Devices are configured.

AT89C5131:    use Monitor in folder T89C51RC2
AT89C5132:    use Monitor in folder AT89C51SND1
AT89C51CC03:  use Monitor in folder AT89C51CC03
AT89C51RD2:   use Monitor in folder AT89C51RD2
AT89C51SND1:  use Monitor in folder AT89C51SND1
AT89C51SND2:  use Monitor in folder AT89C51SND1
T89C51AC2:    use Monitor in folder T89C51CC01
T89C51CC01:   use Monitor in folder T89C51CC01
AT89C51RD2:   use Monitor in folder AT89C51RD2
